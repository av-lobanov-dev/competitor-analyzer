# Архитектура Competitor Analyzer

## Цель

Сервис анализирует сайты конкурентов:
1. Получает сайт конкурента.
2. Открывает сайт через Playwright.
3. GPT анализирует структуру сайта.
4. Playwright собирает товары и цены.
5. PostgreSQL хранит товары, цены и историю изменений.
6. n8n управляет всем процессом.

## Компоненты

### n8n
Главный управляющий процессами.

Задачи:
- запуск анализа сайта;
- запуск Playwright;
- передача данных между сервисами;
- отправка результата клиенту.

### Playwright
Браузерный исполнитель.

Задачи:
- открыть сайт;
- перейти по страницам;
- собрать HTML/текст/цены;
- вернуть данные для анализа.

### GPT
Аналитик.

Задачи:
- понять структуру сайта;
- определить, где категории, товары, цены;
- подготовить карту сайта;
- нормализовать данные.

### PostgreSQL
База данных.

Задачи:
- хранить клиентов;
- хранить сайты конкурентов;
- хранить товары;
- хранить цены;
- хранить историю изменений.
