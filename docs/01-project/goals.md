# Цель проекта

## Название

Competitor Analyzer

## Основная цель

Создать сервис для автоматического сбора и анализа информации о товарах конкурентов.

Клиент передает список сайтов конкурентов. Система должна самостоятельно исследовать сайты, определить структуру каталогов и карточек товаров, собрать информацию и подготовить сравнительные данные для клиента.

## Какие данные необходимо собирать

В зависимости от доступности данных на сайте:

- название товара;
- цена;
- старая цена;
- наличие;
- артикул;
- бренд;
- категория;
- характеристики;
- изображения;
- ссылка на товар;
- дата и время сбора.

## Основные возможности сервиса

1. Добавление клиентов и сайтов конкурентов.
2. Автоматическое исследование структуры сайта.
3. Создание правил сбора данных.
4. Регулярный сбор товаров и цен.
5. Хранение истории цен и наличия.
6. Сопоставление одинаковых товаров разных конкурентов.
7. Анализ изменений.
8. Подготовка отчетов и уведомлений для клиента.

## Основной принцип

Новый сайт конкурента должен подключаться с минимальным количеством ручной настройки.
