# Устаревшая структура Playwright

## Каталог

```text
playwright/app/
```

## Статус

Предварительный статус:

```text
legacy — устаревшая версия
```

Каталог пока не удаляется.

## Почему он считается устаревшим

Текущий Docker Compose монтирует:

```text
./playwright:/app
```

и запускает:

```text
src/queue-runner.sh
```

Следовательно, фактически используется:

```text
playwright/src/
```

Каталог `playwright/app/src/` текущей командой запуска не используется.

## Отличия

Основной проект:

```text
playwright/package.json
Playwright 1.55.0
main: src/worker.js
```

Устаревший проект:

```text
playwright/app/package.json
Playwright ^1.61.1
main: src/index.js
```

Docker-образ также основан на Playwright 1.55.0.

## Состав устаревшей версии

Каталог содержит:

- браузерный модуль;
- обход страницы;
- сохранение результатов;
- подготовку GPT-запроса;
- подготовку карты сайта;
- вспомогательные команды;
- конфигурацию;
- логирование.

## История

Каталог был добавлен первым коммитом:

```text
0ce15ec Initial project setup
```

В последующих коммитах рабочая реализация развивалась в:

```text
playwright/src/
```

## Решение

До удаления необходимо:

1. проверить код подготовки GPT-запросов;
2. определить, используется ли его логика в n8n;
3. перенести полезные функции в актуальную структуру;
4. убедиться, что старый каталог не нужен;
5. удалить каталог отдельным коммитом.

Удалять `playwright/app/` без этой проверки нельзя.
