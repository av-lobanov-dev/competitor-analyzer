# Текущее состояние проекта

Дата актуальности: 2026-07-20

## Проект

Название проекта:

```text
Competitor Analyzer
```

Папка проекта на сервере:

```text
/opt/competitor-analyzer
```

Основная ветка Git:

```text
main
```

## Подтверждённые компоненты

В проекте используются:

- Ubuntu Server;
- Docker;
- Docker Compose;
- PostgreSQL;
- n8n;
- Playwright;
- Node.js;
- GPT;
- Git;
- GitHub.

## Реализованная инфраструктура

- проект размещён на сервере;
- сервисы запускаются через Docker Compose;
- PostgreSQL работает в отдельном контейнере;
- n8n работает в отдельном контейнере;
- Playwright работает в отдельном контейнере;
- Node.js worker подключаются к PostgreSQL;
- репозиторий связан с GitHub;
- изменения отправляются в ветку `main`.
