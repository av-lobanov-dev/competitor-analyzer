## 2026-07-20 — инвентаризация Playwright

### Документация

- задокументирован автоматический запуск Playwright worker;
- задокументирован worker исследования сайтов;
- задокументирован Product Worker;
- описана устаревшая структура `playwright/app`;
- описаны ограничения текущей реализации;
- добавлены исключения резервных копий PostgreSQL.

### Подтверждённая архитектура

- `queue-runner.sh` проверяет очереди каждые 15 секунд;
- `worker.js` обрабатывает задания из `scan_jobs`;
- `product-worker.js` обрабатывает задания из `product_scan_jobs`;
- оба worker используют `FOR UPDATE SKIP LOCKED`;
- каждый worker получает максимум одно задание за запуск;
- `playwright/app` не используется текущим Docker Compose.


В этом документе фиксируются значимые изменения проекта.

Записи располагаются от новых к старым.
