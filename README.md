# Competitor Analyzer

Сервис для анализа конкурентов.

## Цель проекта

Клиент даёт сайты конкурентов. Система собирает товары и цены, объединяет данные по одинаковым товарам и помогает анализировать изменения цен.

## Роли компонентов

- n8n — управляет процессами
- Playwright — открывает сайты и собирает данные
- GPT — анализирует структуру сайта и данные
- PostgreSQL — хранит результаты

## Структура проекта

- playwright/ — браузерные скрипты
- prompts/ — промпты для GPT
- n8n/ — workflows и данные n8n
- postgres/ — данные и backups PostgreSQL
- docs/ — документация проекта
- scripts/ — полезные команды
- data/ — временные данные
- logs/ — логи
